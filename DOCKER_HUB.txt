Usage:
------
This Docker image contains the trained House Prices prediction model
built using MLflow's build-docker command. The image automatically runs
an MLflow model serving endpoint.

Make predictions:
curl -X POST http://localhost:5000/invocations \
  -H 'Content-Type: application/json' \
  -d '{"dataframe_split": {"columns": [...], "data": [[...]]}}'
